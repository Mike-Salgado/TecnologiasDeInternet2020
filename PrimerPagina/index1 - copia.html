¿Que es HTML y Que versiones tiene?

HTML significa lenguaje de marcado de hipertexto, y le permite al usuario crear y estructurar secciones, parrafos,encabezados, enlaces y elementos de cita en bloque para páginas web y aplicaciones

Las versiones de HTML que existen son HTML 2.0, el HTML 3.2, el HTML 4.0, el HTML 4.01 y el HTML 5
